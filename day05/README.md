# Tic Tac Toe (XO Game) with High Scores

This is a terminal-based **Tic Tac Toe (XO)** game implemented in Python.  
You can play:
- **Player vs Player**
- **Player vs Computer**
- View and maintain a **High Score table**

The game automatically saves and updates player scores in `high_score.dat`.
